Please run the results.py file with each .average.csv file to view the result metrics.

eg: python results.py ibce-abcpred-predictions.csv

The results for all testsets combined together i.e. on the pooled dataset are in the 'pooled-all-results.csv' file.

The results for pooled-reduced (homology reduced wrt to iBCE-EL training dataset) are in pooled-reduced-results.csv file.

Please use results for pooled-reduced.py file for viewing results for the pooled reduced file.
